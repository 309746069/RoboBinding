Bind the gap

RoboBinding is a data-binding presentation model framework for the Android platform. RoboBinding helps you write UI code that is easier to read, test and maintain.

provide a pojo presentation model class that exposes the relevant property:

and RoboBinding will provide fine-grained synchronization between view and presentation model.
