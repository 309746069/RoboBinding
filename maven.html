Maven integration

To include RoboBinding to your project dependencies, add the following to your POM:

{% highlight xml %} org.robobinding robobinding 0.8.0-SNAPSHOT {% endhighlight %}

If you're using the AspectJ-powered features of the framework, then you'll need to add both the AspectJ runtime dependency and the AspectJ Maven plugin.

To your dependencies add:

{% highlight xml %} org.aspectj aspectjrt 1.6.11 {% endhighlight %}

To your build plugins, add:

{% highlight xml %} org.codehaus.mojo aspectj-maven-plugin 1.4 process-sources compile org.robobinding robobinding ${maven.compiler.source} ${maven.compiler.target} org.apache.maven.plugins maven-compiler-plugin 2.3.2 ${maven.compiler.source} ${maven.compiler.target} {% endhighlight %}
