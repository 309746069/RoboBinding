Eclipse integration

Needless to say, you will need both Eclipse and the Android Development Tools plugin installed.

If you want to use AspectJ to auto-generate the property change event code in your presentation models (recommended), you will need to install the AspectJ Development Tools (AJDT) plugin.

Once you have it installed, create a new Android project as normal, and then on the project right-click→Configure→Convert to AspectJ Project. This will add the AspectJ natures to your .project file, and you should be good to go.

Also, don't forget to add the RoboBinding.jar to your classpath. Right click on project→Properties→Java Build Path→Libraries→Add JARs. Alternatively for adding the RoboBinding.jar to your dependencies via Maven, see the relevant section.
