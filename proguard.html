ProGuard

RoboBinding requires that all classes which implement ObservableProperties (i.e. presentation models) have their names and public methods left intact. Additionally, we need to preserve all annotations.

To achieve this, add the following directives to your ProGuard configuration:

{% highlight php %} -keep class * implements org.robobinding.property.ObservableProperties { public *** *(...); } -keepattributes *Annotation* {% endhighlight %}

From 0.8.1 we have added a dependency on the guava libraries. At the moment ProGuard doesn't seem to play nicely with guava, so you'll need to the add the following additional line:

{% highlight php %} -dontwarn com.google.common.collect.MinMaxPriorityQueue {% endhighlight %}
